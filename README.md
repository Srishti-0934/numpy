# 📊 NumPy Student Performance Analyzer

A beginner-friendly data analysis project built entirely using **NumPy**. This project demonstrates how to perform statistical analysis on student performance data without relying on libraries like Pandas.

The application reads student records from a CSV file and generates useful insights such as subject-wise statistics, student percentages, rankings, grades, and overall performance analysis using only NumPy.

---------------

## 🚀 Features

- 📂 Load student data from a CSV file using NumPy
- 📈 Calculate subject-wise statistics
  - Mean
  - Median
  - Minimum
  - Maximum
  - Variance
  - Standard Deviation
- 📊 Calculate student percentages
- 🏆 Identify overall topper
- 🥇 Find subject-wise toppers
- 📉 Identify the lowest scorer
- 🎓 Generate student grades
- ✅ Pass/Fail analysis
- 📚 Subject-wise performance insights
- ⚡ Built entirely using NumPy

## 🎯 Learning Objectives

This project is designed to strengthen core NumPy concepts, including:

- Creating and manipulating arrays
- Reading CSV files using `np.genfromtxt()`
- Array indexing and slicing
- Boolean indexing
- Vectorized operations
- Statistical analysis
- Searching and sorting
- Aggregation functions
- Working with multidimensional arrays
